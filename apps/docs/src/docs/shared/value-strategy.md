### Value Strategy

Different strategies handle values during visibility changes, letting you control what happens to the values when their control or group is hidden.

The following strategies are available:

| Strategy            | Effect                                        |
|---------------------|-----------------------------------------------|
| `default` (default) | Sets the specified default value              |
| `last`              | Preserves the last input value                |
| `reset`             | Resets the value using Angular's form API     |

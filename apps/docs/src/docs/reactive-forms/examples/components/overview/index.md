These pages collect the example components used throughout the documentation. Each one is a complete, working implementation of a control, group, array, or block.

Every page shows the component running, the config that drives it, and the full source (`.ts`, `.html`, `.scss`). Read them as a reference for how a component type is built, or copy one as a starting point for your own.

For how the component contract works, see the [Controls](/reactive-forms/guides/controls), [Groups](/reactive-forms/guides/groups), and [Arrays](/reactive-forms/guides/arrays) guides.

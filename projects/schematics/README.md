# @ngx-formbar/schematics

This package contains schematics for ngx-formbar.

## Available Schematics

| Schematic  | Description                                     |
|------------|-------------------------------------------------|
| `ng-add`   | Add ngx-formbar to the project.                 |
| `control`  | Scaffolds a new control and registers it.       |
| `group`    | Scaffolds a new group and registers it.         |
| `block`    | Scaffolds a new block and registers it.         |
| `register` | Searches for all components and registers them. |
